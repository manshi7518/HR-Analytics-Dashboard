# HR Analytics Dashboard

## Project Overview
This project analyzes employee attrition trends and workforce insights using Power BI. The dashboard helps identify key factors affecting employee turnover, such as department, job role, age group, gender, and salary slab.

## Tools Used
- Power BI
- SQL
- Excel

## KPIs
- Total Employees
- Average Age
- Average Income
- Attrition Count
- Attrition Rate

## Dashboard Features
- Department-wise Attrition Analysis
- Job Role-wise Attrition Analysis
- Salary Slab Analysis
- Age Group Analysis
- Gender-wise Attrition Analysis
- Interactive Filters

## Key Insights
- Research & Development department shows the highest attrition.
- Employees aged 26–35 have the highest attrition rate.
- Employees earning less than 5K show higher attrition.
- Laboratory Technicians and Sales Executives have higher turnover.
